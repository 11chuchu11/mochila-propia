a. ¿Quién inventó el sistema de control de versión Git y por qué?

    Fue creado por Linus Torvalds. Debido a que el sistema de control de versiones que utilizaba(BitKeeper) en ese momento no cumplia con todos sus requisitos y que este se habia vuelto de pago decidio crear el suyo propio supliendo todas sus necesidades.  

b. ¿A quién pertenece actualmente Github y por qué?

    Github hoy en día pertenece a Microsoft ya que fue comprado por este el 26 de octubre de 2018 por la suma de 7500 millones de dolares.
